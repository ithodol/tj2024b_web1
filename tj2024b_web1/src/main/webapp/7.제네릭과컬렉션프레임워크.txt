[제네릭 타입]
	1. 정의 : 여러가지 타입에서 동작하라 수 있도록 만든 문법
	2. 목적 : 클래스를 만들 때 변수 타입을 정하지 않고, 인스턴스를 생성할 때 변수 타입을 정한다
		-> 다양한 타입의 데이터를 처리할 수 있다
	3. 사용법 ** < > 사용
		1. 클래스 정의
			(1)
			class 클래스명<제네릭타입명>{
				제네릭타입명 변수명;
			} 
			(2)
			class 클래스명<제네릭타입명A, 제네릭타입B> {
				제네릭타입명A 변수명;
				제네릭타입명B 변수명;
			}
			* 주로 제네릭타입명은 대문자 1개로 작성
		2. 제네릭 타입을 이용한 인스턴스 생성
			(1) 클래스명<타입명> 변수명 = new 생성자명<>();
			(2) 클래스명<타입명, 타입명> 변수명 = new 생성자명<>(); 
			
	4. 주사용처 : 컬렉션프레임워크, ArrayList<>
	5. 특징
		(1) 변수의 유연성
		(2) 코드중복제거
		(3) 잘못된 타입 방지
